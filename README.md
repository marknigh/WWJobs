# Warson Woods Jobs

## Overview

Warson Woods Jobs is a web application built using Vite and Shadcn. It provides a platform for parents and workers to connect for job opportunities. The project leverages modern tools and libraries like React, TypeScript, Firebase, and TailwindCSS for a seamless development and user experience.

## Features

- **Parent and Worker Views**: Separate interfaces for parents and workers.
- **Job Management**: Create, edit, and view job postings.
- **Firebase Integration**: Real-time database and authentication.
- **Responsive Design**: Optimized for desktop and mobile devices.
- **Modern UI Components**: Built with Shadcn and Radix UI.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/vite-shadcn.git
   ```
2. Navigate to the project directory:
   ```sh
   cd vite-shadcn
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
   or
   ```sh
   yarn install
   ```

### Running the Development Server

To start the development server, run:

```sh
npm run dev
```

or

```sh
yarn dev
```

The application will be available at `http://localhost:3000`.

### Building for Production

To build the project for production, run:

```sh
npm run build
```

or

```sh
yarn build
```

The output will be in the `dist` directory.

### Running Tests

To run the test suite, execute:

```sh
npm run test
```

or

```sh
yarn test
```
